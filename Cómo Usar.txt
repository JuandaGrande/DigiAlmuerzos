¡Bienvenido a DigiAlmuerzos!

Gracias por interesarte en nuestro proyecto, DigiAlmuerzos. Nuestra misión es abordar el problema del desperdicio de alimentos en el Colegio Cafam y mejorar la satisfacción de los estudiantes con la oferta alimentaria. A través de la personalización de los almuerzos, buscamos reducir el desperdicio de comida y asegurarnos de que cada estudiante reciba una comida que se adapte a sus necesidades y gustos.

Para comenzar a utilizar DigiAlmuerzos, sigue estos pasos:

Paso 1: Asegúrate de tener Python instalado en tu computadora. Si aún no lo tienes, puedes descargarlo desde el sitio web oficial de Python (https://www.python.org/).

Paso 2: Descarga el archivo comprimido (zip) que contiene todos los archivos necesarios desde nuestro repositorio en GitHub (https://github.com/JuandaGrande/DigiAlmuerzos).

Paso 3: Extrae el contenido del archivo zip en la ubicación de tu elección en tu computadora.

Paso 4: Ejecuta el archivo "requisitos.bat" para instalar las dependencias necesarias. Este archivo se encargará de configurar el entorno de trabajo para DigiAlmuerzos.

Paso 5: Una vez completados los pasos anteriores, estás listo para usar DigiAlmuerzos. Simplemente ejecuta el archivo "autorun.bat" para iniciar el programa.

**Nota Importante:** El archivo "menu.xlsx" debe ser actualizado manualmente para que el programa funcione correctamente, ya que utiliza la fecha actual para leer las opciones de almuerzo.

¡Listo! Ahora puedes disfrutar de un servicio de alimentación personalizado y contribuir a la reducción del desperdicio de alimentos en el Colegio Cafam.
